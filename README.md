## 개요
- 이 프로젝트는 브라우저에서 즐길 수 있는 단순 야구 미니게임입니다.
- 공이 내려올 때 방향과 타이밍을 맞춰 타격하면 점수를 획득합니다.
- 5라운드를 완료하면 최종 점수가 집계됩니다.

## 플레이 방법
- **방향 이동**: 키보드 ← → 화살표로 방망이 위치를 왼쪽/오른쪽으로 변경
- **타격**: Space(스페이스 바)
- **규칙**
  - 공의 진행 방향과 방망이 위치가 일치해야 타격 판정이 발생합니다.
  - 타격 타이밍이 매우 정확하면 "홈런"으로 높은 점수(+20)
  - 타이밍이 근접하면 "파울"로 보통 점수(+10)
  - 놓치면 "스트라이크"
  - 라운드는 총 5라운드이며, 각 라운드마다 공 1회가 진행됩니다.

## 주요 기능
- 라운드 진행 및 점수 집계
- 타격 판정(홈런/파울/스트라이크)
- 시각적 강조(타격 존 하이라이트), 간단한 애니메이션(framer-motion)
- 반응형 키 입력 처리 및 게임 루프 관리

## 기술 스택
- **프론트엔드**: React 19, React DOM
- **애니메이션**: framer-motion
- **번들러/개발 서버**: Vite
- **언어**: TypeScript

## 로컬 실행
사전 준비: Node.js (현대 LTS 권장)

1. 의존성 설치
   ```bash
   npm install
   ```
2. 개발 서버 실행
   ```bash
   npm run dev
   ```
3. 브라우저에서 `http://localhost:3000` 접속
